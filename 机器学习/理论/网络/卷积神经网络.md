## 卷积公式
### 参数

| 符号    | 含义                                                          |
| ----- | ----------------------------------------------------------- |
| $R$   | 被卷积子矩阵中除中心点外其他点相对于中心点的相对坐标集合，定义了卷积的`kernel size`和`dilation` |
| $w$   | 卷积核中的权重                                                     |
| $p_0$ | 被卷积子矩阵中心点的绝对坐标                                              |
| $p_n$ | 被卷积子矩阵中除中心点外其他点相对于中心点的相对坐标                                  |
| $x$   | 被卷积矩阵                                                       |
### 公式
$$y(p_0) = \sum_{p_n\in R} w(p_n) \cdot x(p_0 + p_n)  $$

---
## 参数之间的数量关系

假设特征图长宽相等，卷积核长宽相等
### 参数

| 符号        | 含义                 |
| --------- | ------------------ |
| $L_{in}$  | 输入图像尺寸             |
| $L_{out}$ | 输出图像尺寸             |
| $C_{in}$  | in_channels 输入通道数  |
| $C_{out}$ | out_channels 输出通道数 |
| $K$       | kernel_size 卷积核尺寸  |
| $P$       | padding 填充像素数      |
| $S$       | stride 卷积核移动步长     |

### 计算公式
$$L_{out} = \left\lfloor \frac{L_{in} + 2P - K }{S} \right\rfloor + 1$$
- $C_{out}$与卷积核的数量相同

## 特征图 **feature map**

- **卷积核**对图像进行卷积操作得到的三维矩阵
- 每个像素点可以标识图像中一些特定的特征
---
## 感受野 **receptive field**

- 特征图上某个点能**看到**的输入图像的区域
- 感受野越大，表示该像素点能接触到的原始图像范围就越大，能捕获更全局的信息；越小则表示其包含的特征越趋向于局部和细节
### 增大感受野的方式

- 使用更大的卷积核
- 增加网络深度（越高层的特征图的像素点的感受野通常越大）
- 使用步长更大的卷积层或池化层
- 使用**空洞卷积**
### 减小感受野的方式

- 使用步长小于1 的卷积层
- 使用**池化操作**
